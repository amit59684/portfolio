# Amit Adhikary's Portfolio

Welcome to my personal portfolio website! This project showcases my skills, projects, and contact information. It is designed to provide an overview of my experience as a backend engineer and my proficiency in various technologies.

## Features

- **Home Section**: Introduction with a typing animation.
- **Profile Picture**: Displayed prominently on the home page.
- **Projects Section**: A grid layout showcasing my key projects with descriptions and images.
- **Skills Section**: List of skills with corresponding icons.
- **Contact Section**: Contact form for easy communication.
- **Social Media Links**: Icons linked to my WhatsApp, Instagram, GitHub, and LinkedIn profiles, fixed on the right side of the page.

## Technologies Used

- **HTML**
- **CSS**
- **JavaScript**
- **Python**
- **Django**
- **MySQL**
- **MongoDB**
- **Django REST Framework**

## Projects Included

- **Terminal Based Maze Solver**: Developed a terminal-based application to solve mazes using various algorithms. *(Python)*
- **Sudoku Solver**: Created a tool to solve Sudoku puzzles efficiently. *(Python)*
- **Terminal Based Typing Master**: Built a terminal-based typing practice program. *(Python)*
- **Weather Fetch**: Designed an application to fetch and display weather data from an API. *(Python, API Integration)*

## How to Use

1. Clone the repository: `git clone https://github.com/your-github-username/portfolio`
2. Open `index.html` in your web browser to view the portfolio.
